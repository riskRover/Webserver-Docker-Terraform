🚀 Docker & Terraform Deployment Pipeline 🚀

A streamlined workflow using GitHub Actions, Terraform, and Docker

This pipeline automates the deployment process:

  • Code is pushed to GitHub.
  • GitHub Actions triggers a Terraform plan.
  • Terraform builds and runs the Docker container.
  • Your web server runs seamlessly inside the container.

Developed with ❤️ using HTML, CSS, GitHub Actions, Terraform, and Docker
